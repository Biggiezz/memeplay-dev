## Homepage Game Display

The MemePlay homepage displays a curated feed with specific display rules:

- **One game per template**: The homepage shows only one game per template
- **Highest likes selection**: The displayed game is the one with the highest likes at load time
- **Presentation-layer behavior**: This is a frontend presentation rule, not a database constraint
- **Maximum 11 games**: The homepage displays up to 11 games (one per template)

For detailed technical documentation, see [Homepage Documentation](./homepage.md).
